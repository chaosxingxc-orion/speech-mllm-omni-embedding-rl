## Paper Use

This evidence supports the routing part of the system story:

```text
ASR/text remains the best low-cost path for clean speech, but under dialect or
ASR collapse the direct omni view should be allowed to become the primary route.
Naive fusion is insufficient because bad ASR can pollute the fused ranking.
```

It should not be used to claim that direct omni is universally better than ASR.
The AISHELL-1 row is the explicit negative control against that stronger claim.
